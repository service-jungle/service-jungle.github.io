Роблячи Акт замовлення товару та вказавши в поле попередньо прийнятий пристрій, програма автоматично видалить два акти при видачі гарантії.

Для чого він потрібен?

Якщо ти маєш декілька майстерень або, наприклад сьогодні ти вихідний, а твій колега працює, ти зможеш контролювати все що відбувається дистанційно.

Активуючи бота, ти будеш отримувати сповіщення по прийманню, продажу, видаленню, запису чайових.

В програму можна вписати до трьох акаунтів.

Відкрий всій телеграм та впиши в пошук - @BotFather. Це основний телеграм бот, який створить бота для твоєї програми.

Напиши акаунту @BotFather слово – start. Бот відправить тобі список налаштувань.

Вибери - /newbot – та вкажи ім’я для свого бота.

Далі придумай ім’я посилання з закінченням _bot.

@BotFather створить нового бота та відправить тобі API ключ.

Встанови цей ключ в поле ‘’Токен телеграм-бота:’’ та збережи.

Вище в повідомленні буде також вказано назву бота - t.me __ _bot.

Тепер тобі потрібно дізнатись ID свого акаунту телеграм.

Вписуй в пошук - @my_id_bot, та напиши йому start.

Тепер скопіюй свій ID в поле ‘’Перший ID Telegram:’’, та збережи. Як бачиш, ти можеш зберегти до трьох ID.

Натисни кнопку “Активувати”, та перезавантаж програму.

Тепер єдине що залишилось, це написати своєму боту слово – start. Це захист від спаму.

Чудово, тепер все працює.
